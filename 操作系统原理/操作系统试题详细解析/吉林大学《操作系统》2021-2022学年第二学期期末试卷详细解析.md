# 吉林大学《操作系统》2021-2022 学年春季学期期末试卷（A）详细解析

说明：本卷与另一份 2021-2022 同题 PDF 内容基本一致，差别主要是水印/页数。本文件仍按完整试卷逐题解析。

## 一、名词解释

### 1. 信号量
答案：信号量是用于进程同步与互斥的整型同步工具，只能通过原子 P/V 操作访问。

解析：P 表示申请资源或等待条件，V 表示释放资源或发出条件成立信号。二值信号量用于互斥，计数信号量可表示多个同类资源。

知识点总结：P/V 操作；原子性；互斥；同步；计数信号量。

### 2. 线程
答案：线程是进程内可独立调度执行的基本单位。同一进程内线程共享进程资源。

解析：线程有自己的 PC、寄存器和栈，但共享代码段、数据段、打开文件等。线程切换开销通常小于进程切换。

知识点总结：线程；轻量级进程；共享地址空间；调度单位。

### 3. 地址映射
答案：地址映射是把逻辑地址转换为物理地址的过程。

解析：分页系统用页表把页号映射为块号；分段系统用段表把段号映射为段基址并检查段长。

知识点总结：逻辑地址；物理地址；页表；段表；MMU。

### 4. PCB
答案：PCB 即进程控制块，是操作系统管理进程的数据结构，也是进程存在的唯一标志。

解析：PCB 保存进程标识、状态、调度信息、CPU 现场、内存信息、打开文件和资源占用情况。

知识点总结：进程控制块；进程状态；上下文；调度信息。

### 5. 文件系统
答案：文件系统是操作系统中负责文件命名、目录、存储、访问控制和空间分配的机制。

解析：文件系统向用户提供按名存取，向磁盘提供块分配和元数据维护。

知识点总结：文件；目录；FCB/inode；空间分配；按名存取。

## 二、简答题

### 1. 死锁必要条件
答案：互斥、请求和保持、不剥夺、循环等待。

解析：四个条件同时成立才可能死锁。破坏任一条件可用于死锁预防。

知识点总结：死锁四条件；资源分配图；死锁预防。

### 2. 中断过程
答案：中断请求产生后，CPU 响应中断，保存断点和现场，查中断向量，执行中断服务程序，恢复现场并返回。

解析：核心是“保存-转入-处理-恢复”。现场通常包括 PC、PSW、寄存器等。

知识点总结：中断响应；中断服务程序；中断向量；现场保护。

### 3. 静态分页与请求分页区别，缺页如何处理？
答案：静态分页运行前装入所需页面；请求分页按需装入。访问页不在内存时产生缺页中断，系统调页、必要时置换、修改页表后重新执行指令。

知识点总结：静态分页；请求分页；缺页中断；页面置换。

### 4. 高速缓存与缓冲区区别
答案：Cache 保存近期可能再次访问的数据副本，用于提高访问速度；Buffer 用于协调速度不匹配或传输粒度不一致。

知识点总结：cache；buffer；局部性；I/O 缓冲。

### 5. 文件顺序存取与随机存取区别
答案：顺序存取按当前位置依次读写；随机存取可按偏移直接定位任意位置。

知识点总结：文件访问方式；顺序访问；随机访问；文件指针。

## 三、银行家算法

Need：P1=(3,4,7)，P2=(1,3,4)，P3=(0,0,6)，P4=(2,2,1)，P5=(1,1,0)。Available=(2,3,3)。

### 1. T0 是否安全？
答案：安全。安全序列可为 P4、P1、P2、P3、P5。

解析：Work=(2,3,3)，P4 可完成，Work=(4,3,7)；P1 完成，Work=(6,4,9)；P2 完成，Work=(10,4,11)；P3 完成，Work=(14,4,16)；P5 完成，Work=(17,5,20)。

知识点总结：Need 矩阵；安全序列；银行家算法；死锁避免。

### 2. P2 请求 (0,3,4) 能否分配？
答案：不能。

解析：Request <= Need 成立，但 Available 的 C 只有 3，小于请求的 4，不满足 Request <= Available，因此必须等待。

知识点总结：请求合法性；Available 约束；资源不足等待。

### 3. P4 请求 (2,0,1) 能否分配？
答案：可以。

解析：（2）没有实际分配，仍从 T0 判断。P4 Request <= Need 且 Request <= Available。试分配后 Available=(0,3,2)，P4 Need=(0,2,0)，P4 可完成并释放资源，之后系统仍有安全序列。

知识点总结：试分配；安全性复查；银行家算法三步判断。

## 四、段式存储管理

### 1. 最多多少段？
答案：2^8=256 段。

知识点总结：段号位数；地址结构；段式管理。

### 2. 每段最大长度？
答案：2^16=65536B，即 64KB。

知识点总结：段内偏移；段长上限；二进制地址。

### 3. 地址转换
答案：[0,430] -> 2530；[1,50] 越界中断；[2,30] 缺段中断；[3,70] -> 4070。

解析：段 0 基址 2100，2100+430=2530；段 1 长 40，偏移 50 越界；段 2 不在内存；段 3 基址 4000，4000+70=4070。

知识点总结：段表；段长检查；段基址；越界中断；缺段中断。

## 五、LRU 页面置换

访问串：4，3，2，1，4，3，5，4，3，2，1，5。

### 1. 4 个存储块
答案：缺页序列 4，3，2，1，5，2，1，5；缺页次数 8；缺页率 66.67%。

知识点总结：LRU；缺页率；页面访问串；最近最久未使用。

### 2. 5 个存储块
答案：缺页序列 4，3，2，1，5；缺页次数 5；缺页率 41.67%。

知识点总结：驻留集大小；工作集；局部性；内存块数影响。

### 3. 说明
答案：增加物理块数可降低本题缺页率；LRU 属于栈算法，不出现 FIFO 的 Belady 异常。

知识点总结：Belady 异常；栈算法；页面置换算法比较。

## 六、多级索引文件

块大小 4KB，地址 4B，每个索引块 1024 个地址。

### 1. 最大文件长度
答案：4402345713664B，即 4TB + 4GB + 4MB + 40KB。

解析：数据块数=10+1024+1024^2+1024^3=1074791434，乘 4096B。

知识点总结：多级索引；直接索引；一次/二次/三次间接索引；最大文件长度。

### 2. 2GB 文件实际占用空间
答案：2149584896B。

解析：2GB 文件需 524288 个数据块。索引块为 1 个一次间接块、1 个二次间接根块、511 个二次下级索引块。总块数=524288+513=524801，占用 524801*4096=2149584896B。

知识点总结：索引块开销；实际占用空间；ceil 计算；大文件组织。

## 七、吸烟者问题

### 1. 补全模拟思路
答案：代理员每次放两样物品：烟草+火柴唤醒有纸者，纸+火柴唤醒有烟草者，纸+烟草唤醒有火柴者。

解析：桌面一次只允许放一组物品，能凑齐三样的吸烟者拿走后吸烟，并通知代理员继续。

知识点总结：同步问题建模；生产者-消费者变形；条件同步；互斥访问桌面。

### 2. 计数信号量实现
答案：设置 agent=1，smokerWithPaper=0，smokerWithTobacco=0，smokerWithMatches=0。代理员 P(agent) 后放物品并 V 对应吸烟者；吸烟者 P 自己的信号量，取物、吸烟，最后 V(agent)。

解析：agent 保证桌面互斥；三个吸烟者信号量保证只有条件满足者被唤醒。

知识点总结：计数信号量；条件同步；互斥；P/V 操作；经典同步问题。

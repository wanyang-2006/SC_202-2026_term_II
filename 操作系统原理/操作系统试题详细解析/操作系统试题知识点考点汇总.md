## 高频计算模板
1. 页式地址转换：页号=逻辑地址 div 页大小；页内偏移=逻辑地址 mod 页大小；物理地址=块号*页大小+偏移。
2. 段式地址转换：先查段号；若段号不存在则段号越界；若偏移 >= 段长则越界；若段不在内存则缺段；否则物理地址=段基址+偏移。
3. 银行家算法：先算 Need=Max-Allocation；请求时检查 Request <= Need、Request <= Available；试分配后找安全序列。
4. 页面置换：逐个访问页面，记录内存块状态；未命中计缺页；FIFO 淘汰最早进入者，LRU 淘汰最久未访问者。
5. 索引文件：每个索引块指针数=块大小/指针大小；最大文件数据块数=直接块数+一级可指块数+二级可指块数+三级可指块数；实际占用要加索引块。
6. 磁盘调度：先明确当前磁头、上一位置和移动方向；按算法列服务序列；逐段累加柱面距离；平均寻道长度=总移动量/请求数。

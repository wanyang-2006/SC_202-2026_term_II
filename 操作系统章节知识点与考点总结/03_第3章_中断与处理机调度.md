# 第3章 中断与处理机调度

> 整理依据：`课件/操作系统原理课件 2026/03第三章 中断与处理机调度.pptx`。

## 章节定位

本章连接硬件事件处理和 CPU 分配策略。中断部分要掌握中断源、中断向量、中断嵌套、优先级和处理流程；调度部分要能计算周转时间、带权周转时间、等待时间和响应时间，并比较 FCFS、SJF、SRTN、HRN、HPF、RR、多级队列、反馈队列、EDF、RMS 等算法。

## 核心知识点

1. ==中断的概念==
   - 中断是 CPU 执行当前程序时，由内部或外部事件引起，暂停当前程序转去执行相应处理程序，处理完后再返回原程序的机制。
   - 中断使操作系统能响应 I/O 完成、时钟、异常、系统调用和硬件故障。
   - 自愿性中断通常指系统调用，程序主动请求内核服务。

2. 中断系统
   - 中断源：产生中断的事件或设备。
   - 中断字：记录中断源、状态和屏蔽信息。
   - 中断向量：保存中断处理程序入口地址的表项。
   - 中断嵌套：高优先级中断可打断低优先级中断处理。
   - 中断屏蔽：临时禁止某些中断，保护关键处理过程。
   - 系统栈：保存被中断程序的现场和嵌套中断现场。

3. 中断响应与处理流程
   - 识别中断源。
   - 保护断点和现场，如 PC、PSW、寄存器。
   - 切换到核心态和系统栈。
   - 根据中断向量转入中断处理程序。
   - 执行具体处理，如唤醒等待 I/O 的进程。
   - 恢复现场并返回，必要时触发调度。

4. 调度指标
   - CPU 利用率：CPU 忙碌时间比例。
   - 吞吐量：单位时间完成作业数。
   - 周转时间：完成时间减提交时间。
   - 带权周转时间：周转时间除以运行时间。
   - 等待时间：在就绪队列中等待 CPU 的总时间。
   - 响应时间：从提交请求到首次响应的时间。

5. 处理机调度算法

| 算法 | 基本思想 | 优点 | 缺点 |
| --- | --- | --- | --- |
| FCFS | 先来先服务 | 简单、公平 | 短作业可能等待很久 |
| SJF | 运行时间短者优先 | 平均等待时间小 | 需预知运行时间，长作业可能饥饿 |
| SRTN | 剩余时间最短者优先 | 抢占式 SJF | 切换频繁，长作业不利 |
| HRN | 响应比最高者优先 | 兼顾等待时间和运行时间 | 每次调度需计算响应比 |
| HPF | 优先级最高者优先 | 体现紧急程度 | 低优先级可能饥饿 |
| RR | 时间片轮转 | 适合分时系统 | 时间片过大退化为 FCFS，过小切换开销大 |
| MLQ | 多个固定队列 | 区分前台后台任务 | 队列间可能不公平 |
| FB | 多级反馈队列 | 兼顾交互与批处理 | 参数多，分析复杂 |

6. 调度级别
   - 高级调度：作业调度，从后备队列选择作业进入内存。
   - 中级调度：交换调度，把进程换入或换出内存。
   - 低级调度：进程调度，从就绪队列选择进程占用 CPU。

7. 实时调度
   - EDF：最早截止期优先，动态优先级。
   - RMS：速率单调调度，周期越短优先级越高，静态优先级。
   - 周期任务常用参数：执行时间 C、周期 T、截止期 D。

8. 多处理机调度
   - 自调度：多个处理机从公共就绪队列取任务，简单但队列竞争大。
   - 组调度：相关线程作为一组同时调度，适合并行程序。
   - 负载均衡：避免某些 CPU 空闲而其他 CPU 过载。

## 考点答案与范例

1. 中断与系统调用比较

| 项目 | 中断 | 系统调用 |
| --- | --- | --- |
| 触发来源 | 外设、时钟、故障或异常 | 用户程序主动请求 |
| 目的 | 响应异步或异常事件 | 使用内核服务 |
| 状态变化 | 通常进入核心态 | 由陷入指令进入核心态 |
| 例子 | 磁盘 I/O 完成中断 | open、read、write |

2. HRN 公式

响应比 R = (等待时间 + 要求服务时间) / 要求服务时间 = 1 + 等待时间 / 要求服务时间。

例子：A 需要 10ms、已等 20ms，R=3；B 需要 2ms、已等 2ms，R=2。HRN 选择 A。

3. RR 时间片分析
   - 时间片太大：交互响应变差，近似 FCFS。
   - 时间片太小：响应较快，但上下文切换次数增加，系统开销大。
   - 合理时间片应使大多数交互请求能较快得到响应，又不造成过多切换。

4. 调度计算模板
   - 画时间轴，标出每个进程的开始和完成时间。
   - 周转时间 = 完成时间 - 到达时间。
   - 等待时间 = 周转时间 - 实际运行时间。
   - 带权周转时间 = 周转时间 / 实际运行时间。
   - 平均值 = 所有进程对应指标求和 / 进程数。

## 常见题型与答题抓手

- “给到达时间和服务时间求平均周转”：先画甘特图，再套公式。
- “判断是否抢占”：看到时间片、更高优先级抢占、最短剩余时间，一般是抢占式。
- “实时调度选谁”：EDF 看截止期，RMS 看周期。
- “为什么需要时钟中断”：支持时间片轮转、计时、超时处理和抢占。

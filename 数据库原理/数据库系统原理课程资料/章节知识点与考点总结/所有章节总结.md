# 数据库系统原理 所有章节总结

## 课程知识结构

本课程可分为七组核心内容：

1. 数据库系统概论：基本概念、DBMS 功能、数据抽象、模式/实例、数据模型、数据库语言、系统组件与体系结构。
2. 关系模型与关系代数：关系、键、完整性、选择、投影、并、差、笛卡尔积、重命名、连接、除法、聚集、外连接。
3. SQL：DDL、DML、DCL，基本查询、集合运算、空值、聚集、分组、嵌套查询、视图、完整性约束、授权、函数、过程、触发器。
4. ER 模型：实体、属性、联系、弱实体、映射基数、参与约束、主键、聚集、特化/泛化、ER 图设计与关系模式转换。
5. 关系数据库设计：函数依赖、属性闭包、候选键、正则覆盖、Armstrong 公理、无损连接、依赖保持、BCNF、3NF、多值依赖。
6. 事务：ACID、事务状态、调度、冲突可串行化、视图可串行化、可恢复调度、无级联调度、隔离级别。
7. 并发控制：锁、两阶段锁、严格两阶段锁、强两阶段锁、死锁、多粒度锁、时间戳、验证、多版本、快照隔离。

## 按章节速览

### 第1章 数据库系统概论

- 掌握 DB、DBMS、数据库系统、模式、实例、数据独立性。
- 能比较文件系统和数据库系统。
- 理解三级抽象：物理层、逻辑层、视图层。
- 理解 DDL、DML、SQL、存储管理、查询处理、事务管理。
- 考点：基本概念和简答题。

### 第2章 关系模型与关系代数

- 掌握关系、属性、元组、域、关系模式、关系实例。
- 掌握超键、候选键、主键、外键。
- 熟练写关系代数表达式：`σ`、`π`、`∪`、`-`、`×`、`ρ`、`⋈`、`÷`。
- 掌握聚集、外连接、数据库修改。
- 考点：根据自然语言写关系代数，特别是“所有/全部”对应除法或双重否定。

### 第3章 SQL 基础

- 会写 `create table`、主键、外键、非空约束。
- 熟练掌握 `select-from-where`。
- 掌握 `distinct`、`order by`、字符串匹配、集合运算。
- 掌握 `null` 与三值逻辑。
- 掌握聚集函数、`group by`、`having`。
- 掌握 `in`、`some`、`all`、`exists`、`not exists`、`with`。
- 考点：SQL 查询题、嵌套查询、聚集分组、`not exists`。

### 第4章 SQL 中级

- 掌握内连接、自然连接、左/右/全外连接。
- 掌握视图定义、视图展开、物化视图、可更新视图判断。
- 掌握完整性约束：`unique`、`check`、外键、级联。
- 掌握授权：`grant`、`revoke`、角色。
- 考点：连接查询、视图是否可更新、完整性约束和授权语句。

### 第5章 SQL 高级

- 理解 JDBC、ODBC、嵌入式 SQL 的作用。
- 理解游标处理多行查询结果。
- 掌握函数、存储过程、触发器的基本用途。
- 了解递归查询和固定点计算。
- 考点：函数/过程/触发器概念与简单应用。

### 第6章 ER 模型

- 会识别实体集、属性集、联系集、描述性属性。
- 会标注映射基数和参与约束。
- 会处理弱实体集、递归联系、聚集、特化/泛化。
- 会把 ER 图转换为关系模式。
- 考点：画完整 ER 图、ER 转关系模式，尤其强实体、弱实体、1:1、1:n、m:n、多值属性。

### 第7章 关系数据库设计与规范化

- 掌握函数依赖、属性闭包、候选键。
- 掌握 Armstrong 公理和常用推理规则。
- 掌握正则覆盖计算思想。
- 会判断无损连接和依赖保持。
- 会判定并分解到 BCNF、3NF。
- 了解多值依赖。
- 考点：闭包、候选键、BCNF/3NF 判定与分解、无损连接和依赖保持证明。

### 第17章 事务

- 掌握事务概念和 ACID。
- 掌握事务状态转换。
- 掌握调度、冲突、冲突可串行化、优先图。
- 掌握可恢复调度、无级联调度。
- 掌握隔离级别及并发异常。
- 考点：画优先图判冲突可串行化，判断可恢复/无级联，解释隔离异常。

### 第18章 并发控制

- 掌握 S 锁、X 锁和相容矩阵。
- 掌握 2PL、严格 2PL、强 2PL。
- 掌握死锁检测与恢复。
- 了解多粒度锁、意向锁、时间戳协议、验证协议、多版本和快照隔离。
- 考点：判断锁协议、判断死锁、区分 2PL/严格 2PL/强 2PL。

## 复习课件给出的考试范围

- 第 1 章：基本概念。
- 第 2 章：关系代数表达式，基本运算、附加运算、扩展运算。
- 第 3、4、5 章：SQL 语句，DDL、DML、DCL，集合运算、空值、聚集函数、`having`、`not exists`、嵌套子查询、视图、可更新视图、完整性约束、函数、过程、触发器。
- 第 6 章：ER 模型设计，关系模型转换，完整 ER 图；实体集、弱实体集、属性集、联系集、聚集、描述性属性、映射基数、参与约束、主键。
- 第 7 章：规范化，函数依赖、属性闭包、正则覆盖、多值依赖概念、BCNF/3NF 判定与分解、Armstrong 公理、无损连接和保持依赖证明。
- 第 17 章：事务概念、模型、ACID、冲突/视图可串行化、隔离性和原子性、可恢复调度、无级联调度。
- 第 18 章：两阶段锁、严格两阶段锁、强两阶段锁、死锁检测与恢复。

## 高优先级复习清单

1. 能独立写关系代数表达式，特别是除法和“全部”问题。
2. 能独立写 SQL 查询，特别是嵌套查询、`not exists`、分组聚集和 `having`。
3. 能画 ER 图并转换为关系模式。
4. 能求属性闭包、候选键、正则覆盖。
5. 能判断 BCNF/3NF 并完成分解。
6. 能证明无损连接和依赖保持。
7. 能画优先图判断冲突可串行化。
8. 能判断可恢复调度和无级联调度。
9. 能判断锁调度是否满足 2PL、严格 2PL、强 2PL。
10. 能画等待图判断死锁并说明恢复策略。

## 答题策略

- 概念题：写“定义 + 作用 + 例子/对比”。
- 关系代数题：先找涉及关系，再找连接条件，最后处理投影、差、除法。
- SQL 题：先写核心 `from` 和连接，再写 `where`，最后补聚集、分组、排序。
- ER 题：先实体，后属性，再联系，最后标基数、参与、主键。
- 规范化题：先求闭包和候选键，再判范式，最后按算法分解。
- 事务题：列冲突、画图、看环。
- 锁题：逐事务检查加锁/解锁阶段，死锁题画等待图。
